Zorgaanbieder identificatie ()
VEREISTE - Zorgaanbieders worden geïdentificeerd met een landelijk uniek nummer VERANTWOORDELIJKHEID - Dossierhouder en dossierraadpleger/-ontvanger dienen elkaar te kunnen identificeren. INVULLING TWIIN - URA (UZI-Register Abonneenummer) TOELICHTING - Het URA-nummer is het breedst toepasbare landelijke stelsel om zorgaanbieders te identificeren. Het URA-nummer is verplicht maar daarnaast mogen ook andere identificatienummers gebruikt worden.
