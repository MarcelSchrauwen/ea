Adressering [plateau 2] ()
Communicerende partijen moeten hun eigen adresseringsmechanisme gebruiken om uit te wisselen. Dit kan van alles zijn, van hard coded IP-adressen tot een eigen adresseringsdienst. Twiin stelt hieraan geen eisen. Deze tekst is anders dan plateau 2 maar lijkt hetzelfde te zeggen.
