Twiin afspraak ()
Archimate element - Contract A Contract represents a formal or informal specification of an agreement between a provider and a consumer that specifies the rights and obligations associated with a product and establishes functional and non-functional parameters for interaction. In het geval van Twiin kunnen we deze denk ik vertalen in: Een Twiin afspraak bevat een specificatie van een overeengekomen onderwerp, gemaakt binnen het Twiin bestuur, Twiin community of voortkomend uit wet- en/of regelgeving. Aggregation relatie Een Twiin afspraak kan en zal in de meeste gevallen een relatie hebben met een andere Twiin afspraak. Dit maakt dat afspraken een onderlinge afhankeljkheid van elkaar hebben. Een Twiin afspraak kan bijvoorbeeld afhhankelijk zijn van een landeljke afspaak. Relization Binnen afspraken moeten principes, constraints, en requirements zijn opgenomen om het nakomen van een afspraak meetbaar/ aantoonbaar te maken. Een aspraak zonder één van deze drie heeft eigenlijk geen betekenis.
