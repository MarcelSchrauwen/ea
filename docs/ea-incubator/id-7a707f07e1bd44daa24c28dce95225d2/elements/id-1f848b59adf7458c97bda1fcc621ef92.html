Twiin afsprakenstelsel ()
Het Twin afsprakenstelsels bevat meerdere afspraken. Het Twiin afsprakenstelsel kan gezien worden als een publieke voorziening. Zo'n voorziening bevat minimaal een duidelijke specificatie de voorziening zelf, hoe het bestuurd wordt, en hoe de ondersteuning is vormgegeven.
