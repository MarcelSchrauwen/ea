Logging [plateau 3] ()
Logging en monitoring, conform landelijke afspraken in lijn met NEN 7510, NEN 7512 en NEN 7513. Deelnemende zorgaanbieders voldoen aan de landelijke afspraken op het gebied van logging en bijbehorende procedures voor monitoring en controle van deze logging.
